# Browser Extension Development Examples

A collection of browser extension examples demonstrating progressive development approaches, from basic vanilla JavaScript to modern toolchains.

## Examples

1. [Introduction to Basic Concepts](https://blog.rxliuli.com/p/d324c4840f4c46c2ac5c4af56207905e/) - A minimal extension built with vanilla JavaScript that hides the AI Mode button on Google Search.

## Scripts

```bash
pnpm init-all      # Initialize all examples
pnpm test-all      # Run tests for all examples
```
